1. Install NodeJS
2. Buka cmd tulis perintah "node --v", kalo muncul versinya berarti berhasil
3. Extract here sipuma
4. Nyalain MySQL XAMPP
5. Buka vscode, open folder ke sipuma
6. Buka terminal
7. Tulis perintah :
	> npm i --legacy-peer-deps
	> npx prisma generate
	> npx prisma db push
	> npx prisma db seed
	> npm run dev
8. done, login pake username: admin, password : admin123, buat masuk sebagai admin
